# Retail Analytics using Databricks SQL

## Overview

This project analyzes retail transaction data using Databricks SQL.

## Objectives

- Explore transaction data
- Identify top-selling products
- Analyze revenue drivers
- Evaluate payment method usage

## Skills Demonstrated

- SQL
- Aggregations
- GROUP BY
- ORDER BY
- Data exploration
- Business analytics

## Example Insight

Golden Gate Ginger generated the highest revenue ($11,595) and also had the highest units sold (3,865), indicating that revenue leadership was primarily driven by sales volume rather than pricing.
